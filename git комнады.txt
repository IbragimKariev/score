1) git status

2) git add . - вводится всегда

3) git commit -m "add waxom files" - пишем свое название вместо waxom

4) git push - отправить

5) git pull - принять

ugeday.github.io/Waxom/default.html - просто вводишь в поле и откроется ссылка на ваш проект, следим за индексом букв

.gitignore - создается лишь в редакторе, игнорирует файлы при отслеживании и добавлении оных на гитхаб
